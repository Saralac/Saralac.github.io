在创建博客时遇到的一些问题

1.教程中的npm命令安装hexo时，在控制命令符里更容易实现，快捷打开方式：<开始>-输入cmd，回车。即可打开控制命令符
2.当找不到git部署时，需要安装插件。
命令为npm install hexo-deployer-git --save
需要在根目录下面执行
3.更换主题时，要看清楚部署信息，有些主题需要特殊的依赖环境
4.一些命令需要进入对应的文件夹实现，cmd中进入文件夹方法为cd <文件夹名>
5.配置文件时不能漏掉冒号：后面的空格，空格在这里相当于等号，没有空格就会报错
6.init或者clone操作偶尔会报错，第一次操作就失败了不要直接就放弃，也许一模一样的操作再执行一遍就成功了
